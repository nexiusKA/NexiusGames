# NexiusGames

**Nexius Games** is the central hub for all games made by [nexiusKA](https://github.com/nexiusKA).

The hub page (`index.html`) features an epic dark/neon gaming aesthetic and links to every game repository:

| Game | Repository |
|------|-----------|
| 🌀 Catch the Chaos | [nexiusKA/Catch-the-Chaos](https://github.com/nexiusKA/Catch-the-Chaos) |
| 🔵 Marble Speed Run | [nexiusKA/Marble-Speed-Run](https://github.com/nexiusKA/Marble-Speed-Run) |
| 💨 Tower Fart Defense | [nexiusKA/TowerFartDefense](https://github.com/nexiusKA/TowerFartDefense) |
| 🤖 Beware of the Robots | [nexiusKA/BewareOfTheRobots](https://github.com/nexiusKA/BewareOfTheRobots) |

Open `index.html` in any browser, or enable **GitHub Pages** (Settings → Pages → Deploy from branch `main`, root folder) to host it live.
